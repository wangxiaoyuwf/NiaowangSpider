# NiaowangSpider
从鸟网上爬取数据，把963条数据翻译成中文

操作步骤

1. 从网上下载aiy_birds_V1_labelmap.csv（https://tfhub.dev/google/aiy/vision/classifier/birds_V1/1）

2. 运行程序

3. 结果就会成单独的一列写入到aiy_birds_V1_labelmap_amended.csv
